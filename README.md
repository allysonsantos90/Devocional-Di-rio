# Devocional Diário | Daily Devotional
🇧🇷 Português
Bem-vindo ao Devocional Diário, um espaço de reflexão e renovação espiritual baseado no livro "New Morning Mercies: A Daily Gospel Devotional" de Paul David Tripp.
Cada devocional foi pensado para começar o dia com o coração voltado ao Evangelho, lembrando das misericórdias renovadas de Deus a cada manhã.

"As misericórdias do Senhor são novas a cada manhã." — Lamentações 3:23


🇺🇸 English
Welcome to Daily Devotional, a space for spiritual reflection and renewal based on the book "New Morning Mercies: A Daily Gospel Devotional" by Paul David Tripp.
Each devotional is designed to start the day with a heart focused on the Gospel, remembering God's mercies that are renewed every morning.

"His mercies are new every morning." — Lamentations 3:23
